## Run examples

**Python env**
```
python -m venv venv
source venv/bin/activate
pip install -r requirements.txt 
```

**Setup .envs**
```
cp client/template.env client/.env
nano client/.env
```
```
cp oracles/template.env oracles/.env
nano oracles/.env
```
